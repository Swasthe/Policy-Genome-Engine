# Policy Genome Engine

## Overview
This project is an AI-powered policy analysis system that compares new policies with existing national policies to predict their success and transferability.

## Features
- Semantic policy similarity
- Transferability prediction
- Risk indicator
- Policy strength score
- Explainable AI insights

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py

## Tech Used
- Python
- Streamlit
- Sentence Transformers
- Pandas

## Dataset
Curated Indian national policies from government and academic sources.

## Hackathon Prototype
This project demonstrates a functional AI decision-support system for policymaking.
